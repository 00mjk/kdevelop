2. Installation

2.1 How to obtain KDevelop

KDevelop can be found either on the   KDE Applications page  or on the KDevelop homepage.
    There, we offer weekly snapshots of the KDevelop CVS repository for those who want to stay up to date with KDevelop.

2.2 Requirements

In order to successfully compile and use KDevelop, you need following programs and libraries.

gcc/g++ 2.7.2.1 (or compatible)
perl 5.004
autoconf 2.12
automake 1.2

Qt 1.42
KDE 1.1
KDE-SDK from 31.08.98 or newer
(  with sgmltools 1.0  )
glimpse

KDevelop was tested under SuSE Linux  5.2 on an AMD K6 200, 64MB RAM and under FreeBSD 3.0-Release on an Intel 200 MMX, 128MB RAM.

2.3 Compilation and installation

In order to compile and install KDevelop on your system, type the following in the base directory of the KDevelop distribution:
% ./configure
% make
% make install
Since KDevelop uses
autoconf
you should have not trouble compiling it.

In order to compile the KDevelop CVS snapshot, type the following :

            % make -f Makefile.cvs
            % ./configure
            % make
            % make install
