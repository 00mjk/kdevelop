¡ Bienvenido al KDevelop !

¡ Bienvenido al Entorno Integrado de Desarrollo KDevelop !

El EID KDevelop ha sido diseñado por el equipo de KDevelop para entregar a los desarrolladores de C/C++ un entorno fácil de utilizar. Con todas las características que la versión actual contiene, todo el conocimiento sobre programación del equipo ha sido recogido para juntar un conjunto de manuales que puedan ayudar en el uso del EID tanto a principiantes como a expertos. Esperamos que este sea el entorno que necesita y que le guste para crear aplicaciones que contribuyan al Entorno de Escritorio K (KDE) y a la comunidad del software libre, así como para cualquier propósito comercial.

Actualmente, la documentación del KDevelop se distribuye en cuatro manuales llenos de texto informativo, claro, y bien escrito que le permitirán empezar a desarrollar sus aplicaciones, incluyendo un manual extra para el el desarrollo de aplicaciones para el próximo KDE2:

El Manual de Usuario de KDevelop

Guía de referencia para el Entorno Integrado de Desarrollo KDevelop para sistemas Unix, versión 1.0.

El Manual de Usuario de KDevelop contiene una completa descripción de la funcionalidad del EID. Además, le dará una descripción sobre los proyectos en general y sobre la forma en que se configuran utilizando herramientas de desarrollo estándar de GNU.

El Manual de Programación de KDevelop

Guía de usuario para el diseño de aplicaciones en C++ para el Entorno de Escritorio K (KDE) con el EID KDevelop, versión 1.0.

El Manual de Programación cubre todos los temas que están relacionados con el uso real del EID. Contiene descripciones de los tipos de proyectos disponibles explica los elementos básicos de las aplicaciones KDE y Qt, además de una completa descripción del código fuente generado por el asistente de aplicaciones del KDevelop. Lo que es más, se describen varios pasos del proceso de desarrollo, como el escribir la documentación y las opciones de instalación para sus proyectos de acuerdo al Estándar de Sistema de Ficheros KDE.

Tutoriales de Aplicaciones KDE

Tutoriales de aplicaciones KDE para el Entorno Integrado de Desarrollo KDevelop. Una colección de tutoriales sobre la creación de aplicaciones KDE por medio de ejemplos. También describe como construir los tutoriales de ejemplo incluidos con la biblioteca Qt.

Los tutoriales de aplicaciones KDE contienen instrucciones didácticas, tanto para principiantes como para desarrolladores avanzados, para aprender a diseñar aplicaciones KDE por medio de proyectos de ejemplo. Comenzando con la creación de los tutoriales incluidos con la documentación de la biblioteca Qt, el lector es guiado hasta ejemplos con Qt y hasta la creación de proyectos de aplicaciones KDE. Muchos temas a los que se hace referencia en el Manual de Programación del KDevelop son explicados con más detalle mediante el uso de ejemplos.

Guía de Referencia de la Biblioteca KDE

Guía de referencia para el diseño de aplicaciones en C++ para el Entorno de Escritorio K (KDE).

La guía de referencia contiene información estructurada sobre el marco de aplicaciones KDE y cubre temas como el manejo de eventos, señales/ranuras, los elementos de IGU (Interfaz Gráfica de Usuario) proveidos, y el uso de los diálogos estándar para la introducción de datos por parte del usuario. Lo que es más, contiene información sobre los problemas de portabilidad a Qt 2.0 / KDE 2 relacionados con el marco de las aplicaciones KDE.

Guía de Desarrollo para el KDE 2

Una colección de documentación ya existente sobre el uso de la API del KDE 2, incluyendo información sobre KOM/OpenParts por varios autores del KDE.

Este manual es una colección de artículos ya existentes sobre el desarrollo para KDE 2 que han sido escritos por varios de los autores del KDE. En general, cubre las nuevas facilidades y tecnologías que el KDE 2 ofrecerá a los desarrolladores, además de discutir temas relacionados con su implementación.
