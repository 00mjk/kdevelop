Добро Пожаловать в KDevelop !

Добро пожаловать в KDevelop - Интегрированную Среду Разработки(IDE) !

KDevelop IDE был спроектирован командой разработчиков KDevelop для того, чтобы предоставить программистам легкую в использовании среду разработки программ на C/C++. Помимо различных свойств(средств), текущая версия содержит полный набор знаний о программировании, которые были собранны командой и добавлены в состав документации, чтобы облегчить использование различных частей IDE как начинающим, так и профессионалам. Мы надеемся предложивать вам именно ту IDE, в которой вы нуждаетесь и захотите использовать для создания собственных приложений, которые вы можете предоставить для K Desktop Environment и сообщества свободного программного обеспчения или использовать в коммерческих целях.

На данный момент документация по KDevelop содержит четыре книги, полные хорошо написанным, информативным и понятным текстом, которые позволят вам начать разработку ваших приложений. Сюда также входит дополнительная книга по основам разработки приложений для ожидающегося в скором времени KDE 2:

Руководство пользователя по KDevelop

Справочное руководство по KDevelop - Интегрированной Среде Разработки Приложений для Unix систем, версия 1.0.

Справочное руководство по KDevelop содержит полное описание всех доступных функций IDE. Дополнительно оно предоставит вам развернутую информацию о проектах вообще и об использовании стандартных средств разработки проекта GNU для их создания.

Руководство по программированию в KDevelop

Руководство пользователя по проектированию C++ приложений для KDE с помощью IDE KDevelop, версия 1.0.

Руководство по программированию затрагивает все темы так или иначе связанные с использованием IDE. Оно содержит описание типов проектов и основы построения приложений Qt и KDE, а также полное описание исходных текстов, создаваемых мастером приложений KDevelop. В дальнейшем рассматриваются различные шаги процесса разработки, такие как разработка документации и установочные настройки для файлов проекта в соответствии с системным стандартом KDE на файлы.

Учебник по разработке приложений для KDE

Учебник по разработке приложений для KDE с помощью IDE KDevelop - набор учебных пособий, описывающих на примерах процесс создания KDE приложений и процесс сборки примеров, поставляемых с библиотекой Qt.

Учебник по разработке приложений для KDE содержит объемные инструкции для новичков и профессиональных разработчиков, рассказывая о проектировании Qt и KDE приложений на основе проектов-примеров. Справочное руководство по библиотекам KDE

Справочное руководство по проектированию C++ приложений для KDE.

Справочное руководство содержит структурированную информацию о каркасе приложения KDE и затрагивает такие темы, как обработка событий, сигналы/слоты, поставляемые визуальные элементы и использование стандартных диалогов для ввода пользователем необходимой информации. Также оно содержит информацию об известных особенностях Qt 2.0 / KDE 2.0 по отношению к стандартному каркасу KDE приложения.

Руководство по разработке приложений для KDE 2

Набор существующей на данный момент документации по использованию KDE 2 API, включая вопросы по KOM/OpenParts, различных авторов.

Это руководство является набором уже существующих статей, написанных различными авторами и касающихся разработки приложений для KDE 2. Они в основном описывают новые способности и технологии, предлагаемые разработчикам в KDE 2, а также их реализации.
