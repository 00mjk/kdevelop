2. Установка

2.1 Как получить KDevelop

KDevelop можно найти на    Страничке приложений KDE  или на Домашней страничке KDevelop

2.2 Требования

      Для успешной компиляции и использования KDevelop Вам потребуются следующие программы и библиотеки:

      gcc/g++ 2.7.2.1 (или совместимый)
      perl 5.004
      autoconf 2.12
      automake 1.2

      Бибилиотека Qt 1.4
      KDE 1.1
      KDE-SDK от 31.08.98 ли более новая
       (  c sgmltools 1.0  )

      KDevelop тестировался под SuSE Linux  5.2 на AMD K6 200, 64MB RAM и под SuSE Linux 5.3 на Intel 200 MMX, 128MB RAM.

2.3 Компиляция и установка

Для компиляции и установки KDevelop на вашей системе, перейдите в основную директорию исходных текстов KDevelop и отдайте следующие команды: 
% ./configure
% make
% make install
Так как KDevelop использует 
autoconf
