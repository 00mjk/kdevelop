FEHLER:

 

Die C/C++ Referenz ist nicht installiert.  Bitte holen Sie sich das Paket "c_cpp_reference-1.0.tar.gz" von einem der ftp-Mirrors von KDevelop. Eine entsprechende Liste von solchen Web-Servern finden Sie auf:

www.kdevelop.org

Führen Sie folgende Schritte aus, um die Referenz zu installieren:

  1. Wenn Sie eine schon installierte alte Version von KDevelop haben, ist es empfohlen, eine möglicherweise früher schon installierte alte Referenz, die nicht korrekt funktioniert, zu löschen (als root). Sie können solche 2 Unterverzeichnisse C und CPLUSPLUS in share/doc/HTML/ihre-sprache/kdevelop/reference des globalen KDE-Verzeichnisses und im entsprechenden englischen Unterverzeichnis finden. Löschen Sie beide.
  2. Entpacken Sie die neu heruntergeladene Datei mit: "tar xvzf c_cpp_reference-1.0.tar.gz"
  3. Gehen Sie in das so neu erstellte Verzeichnis: "cd c_cpp_reference-1.0"
  4. Machen Sie das Paket für die Installation fertig: "./configure; make"
  5. Installieren Sie das Paket (als root): "make install"
  6. Starten Sie KDevelop neu.
