Introduction

Welcome to KDevelop.

With KDevelop you have a first-class Integrated Development Environment especially designed for C/C++ application development and with enhanced features for KDE 2.1 Development. KDevelop helps you create and maintain your projects, develop user interfaces in conjunction with TrollTech´s Qt Designer and helps you translate your application in over 40 languages to give your customers and users the best possible usage in their native language. Your IDE also offers you the best access to your sourcecode via the Classbrowser and to your API documentation through the integrated documentation browser.

Simply start your first project with selecting  "New Project" from the "Project" entry in the menubar. The following wizard will guide you through creating your initial project of your choice. If you want to work with KDevelop on an already existing project, select   "Generate Project File..." in the "Project" menu. To open a project file select  ("Open Project") from the toolbar.
